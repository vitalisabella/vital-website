# VITAL Student Organization Website

A comprehensive web platform for VITAL, a student organization focused on mentorship and growth.

## Features

- **Home Page**: Hero section with live announcements
- **About**: Organization mission and values
- **Team**: Meet team members with photos and bios
- **Student Survey**: Feedback on growth journey before/during/after VITAL
- **Resources**: Learning materials organized by category
- **Team Leader Application**: For members looking to advance to leadership roles
- **Admin Panel**: Full control to manage content, announcements, team, and applications

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Visit http://localhost:3000
